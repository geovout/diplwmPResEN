Environmental Crisis & Technology

The environmental crisis is now one of the biggest challenges humanity is facing and it directly affects man and his daily life. Meeting the challenges shall comprise the identification and recording of essential environmental parameters concerning the natural degradation. The technological achievements of the last two centuries have created new prospects of interaction between man and the environment which he is living in it. The participatory systems enhance new perspectives in scientific research through expanded use of technology.

Environmental Paramerters

The studied variables are: relative Humidity και mean Temperature of the air, Air pollution (through CO & CH4), Light pollution and (experimentally) Sound pollution.
The measurements record the state of the environment in an area closer to the recording devices.
The variables are largely determined by technical limitations.

Components

The project involves the experimental implementation of a system which records and monitors elements of the environment through the combined application of as IoT και Web GIS.
The system aims at qualitative assessment of the in-situ environmental conditions and inference from geographic clustering from phenomena which adversely affect it.

Arduino Yun

Arduino Arduino is an open source hardware platform which enables to create automation applications based on easy to use hardware and software. It features a plethora of extensions (modules and sensors) and customization abilities.
Arduino Yun model is used in this study, because it provides the required capabilities for the project.

Android Application

Android platform supports portable devices like smartphones and tablets. Is the most popular operating system platform for both portable and desktop computer devices.
The application is developed with Apache Cordova through standardised web technologies (HTML5, CSS3, and JavaScript).
Cordova supports Ionic framework which used for front-end developing for Android and IoS based on AngularJS.

Web GIS

A Web Map offers capabilities of a supervisory medium which interactively displays spatial information. Cartography offers plenty geo-visualisation methods which are widely exploited with web mapping applications. Web maps offer interaction between the user and provided information.

Recording Device

The experimental recording device is consisted from 5 sensors for environmental observation, which produce 7 variables. It is based on Arduino Yun microcontroller serving fixed and portable operation. It is powered by external source, battery unit (power bank) for portable application and electricity supply for fixed. Measures are transfered through Bluetooth protocoll to an Android device and distributed to a server for further use in a Database system.

Data Distribution

The management and storage of the recordings consist the fundamental elements of the proposed Sensor Network.
Particularly for the purpose of the study a Client - Server schema has been developed utilizing Cloud Computing technologies.

web map

The chosen method of cartographic representation is Heat Map because it provides an optimal visualisation of environmental quality on any given location through variable clustering.
Heat maps act as an environmental assessement system, synchronizing technical sizes and spatial position.

Conclusions

  • The proposed system consists an automated method of monitoring the environmental quality
  • The system supplies the network from many different devices (nodes) simultaneously
  • It requires little resources
  • It provides quantitative and qualitative assessment capabilities
  • It has many hardware extensions capabilities
  • The derived variables offer dynamic represantation capabilities on a Web GIS environment
  • Generalizations occurred that caused measurement accuracy and hardware - software compatibility issues
